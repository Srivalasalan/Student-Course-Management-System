# 🎓 Student Management System

A full-stack Student Management System built using Spring Boot and PostgreSQL.

## 🚀 Features

- Add Student
- Update Student
- Delete Student
- Get Student by ID
- Get All Students
- Assign Course to Student
- Global Exception Handling
- DTO Layer Implementation
- Clean REST API Architecture
- Frontend Integration using HTML, CSS, JavaScript (async/await)

## 🛠️ Tech Stack

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Hibernate
- REST APIs
- HTML, CSS, JavaScript
- Git & GitHub

## 🏗️ Project Architecture

Controller → Service → DAO → Repository → Database

Includes:
- Proper layering
- DTO mapping
- Custom Response Structure
- Exception handling using @ControllerAdvice

## 🧠 Concepts Practiced

- One-to-Many Relationship
- DTO Conversion
- Stream API
- ResponseEntity
- Global Exception Handling
- RESTful API Design
- Frontend-Backend Integration

## 📂 How to Run

1. Clone the repository
2. Configure PostgreSQL in `application.properties`
3. Run the Spring Boot application
4. Open: http://localhost:8080/index.html
